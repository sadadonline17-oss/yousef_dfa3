# دليل الاختبار - نظام روابط الدفع الحكومية

## 🧪 كيفية اختبار التطبيق

### 1️⃣ اختبار المنشئ

#### الخطوة 1: الوصول للمنشئ
```
1. افتح التطبيق من الصفحة الرئيسية
2. اضغط على "الخدمات" أو انتقل لـ /services
3. اختر دولة من القائمة (مثال: السعودية)
4. ستظهر الخدمات المتاحة بما فيها:
   - سداد (للسعودية)
   - بنفت (للبحرين)
   - كي نت (للكويت)
   - مال (لعُمان)
   - جيوان (للإمارات)
   - بوابة الدفع الحكومي (لقطر)
```

#### الخطوة 2: فتح المنشئ
```
1. اضغط على أحد الخدمات الحكومية
2. سيفتح منشئ روابط الدفع المستقل
3. تحقق من:
   ✅ ظهور الشعار الرسمي
   ✅ الألوان الصحيحة للخدمة
   ✅ الخطوط العربية الواضحة
   ✅ جميع الحقول ظاهرة
```

#### الخطوة 3: إنشاء رابط اختبار
```
أدخل البيانات التالية:

اسم العميل: محمد أحمد السعيد
رقم الجوال: 0501234567
البريد الإلكتروني: test@example.com
المبلغ المطلوب: 500
رقم الفاتورة: INV-001
وصف الدفع: فاتورة اختبارية

ثم اضغط "إنشاء رابط الدفع الآمن"
```

#### الخطوة 4: نسخ الرابط
```
بعد إنشاء الرابط:
1. تحقق من ظهور رسالة النجاح ✅
2. انسخ الرابط باستخدام زر "نسخ الرابط"
3. تحقق من أن الرابط بالصيغة:
   https://[domain]/pay/[id]?service=[serviceKey]&country=[countryCode]&amount=[amount]&currency=[currency]&method=card&provider=[PROVIDER]
```

### 2️⃣ اختبار تدفق الدفع الكامل

#### الخطوة 1: صفحة بيانات المستلم (PaymentRecipient)
```
1. افتح الرابط المنسوخ في نافذة جديدة
2. تحقق من:
   ✅ ظهور Header بشعار الخدمة الحكومية
   ✅ الألوان الرسمية للخدمة (مثال: برتقالي لسداد)
   ✅ Carousel يعرض صور الخدمة
   ✅ عنوان الصفحة: "معلومات المستلم"
   ✅ عرض المبلغ المطلوب بوضوح
   ✅ جميع الحقول ظاهرة:
      - الاسم الكامل
      - البريد الإلكتروني
      - رقم الهاتف
      - العنوان السكني
   ✅ زر "متابعة للدفع" ظاهر
   ✅ رسالة الأمان والحماية
```

#### الخطوة 2: إدخال بيانات المستلم
```
أدخل البيانات التالية:

الاسم الكامل: أحمد محمد العلي
البريد الإلكتروني: ahmed@test.com
رقم الهاتف: +966501234567
العنوان السكني: الرياض، حي النخيل، شارع الملك فهد

ثم اضغط "متابعة للدفع"
```

#### الخطوة 3: صفحة تفاصيل الدفع (PaymentDetails)
```
تحقق من:
✅ عرض تفاصيل الشحنة (إن وجدت)
✅ ملخص الدفع يعرض:
   - الخدمة: [اسم الخدمة الحكومية]
   - المبلغ الإجمالي: [المبلغ بالعملة]
✅ عرض طريقة الدفع المختارة:
   - 💳 الدفع بالبطاقة
   أو
   - 🏦 تسجيل دخول البنك
✅ زر "متابعة للدفع" ظاهر
✅ الثيم الحكومي مطبق
```

#### الخطوة 4: إدخال بيانات البطاقة (PaymentCardInput)
```
تحقق من:
✅ نموذج إدخال البطاقة آمن
✅ الحقول الموجودة:
   - اسم حامل البطاقة
   - رقم البطاقة (16 رقم)
   - شهر الانتهاء
   - سنة الانتهاء
   - CVV (3 أرقام)
✅ التحقق التلقائي من رقم البطاقة (Luhn Algorithm)
✅ عرض نوع البطاقة (Visa, Mastercard, Mada)
✅ الثيم الحكومي مطبق

أدخل بيانات بطاقة اختبارية:
- اسم حامل البطاقة: AHMED ALI
- رقم البطاقة: 4111 1111 1111 1111
- شهر الانتهاء: 12
- سنة الانتهاء: 25
- CVV: 123
```

#### الخطوة 5: التحقق بـ OTP (PaymentOTP)
```
تحقق من:
✅ صفحة إدخال OTP واضحة
✅ 6 خانات لإدخال الرمز
✅ العد التنازلي يعمل (3 دقائق)
✅ رسالة "تم إرسال رمز التحقق"
✅ خيار "إرسال الرمز مرة أخرى"
✅ الثيم الحكومي مطبق

أدخل OTP اختباري: 123456
```

#### الخطوة 6: الإيصال النهائي (PaymentReceipt)
```
تحقق من:
✅ رسالة "تم الدفع بنجاح" ✅
✅ عرض تفاصيل الدفع:
   - رقم العملية
   - التاريخ والوقت
   - المبلغ المدفوع
   - الخدمة
   - طريقة الدفع
✅ أزرار:
   - 🏠 العودة للرئيسية
   - 📥 تحميل الإيصال
   - 📤 مشاركة
✅ الثيم الحكومي مطبق
✅ شعار الخدمة ظاهر
```

### 4️⃣ اختبار Responsive

#### على الهاتف المحمول 📱
```
1. افتح Chrome DevTools (F12)
2. اضغط على Toggle Device Toolbar (Ctrl+Shift+M)
3. اختر iPhone 12 Pro
4. تحقق من:
   ✅ جميع العناصر ظاهرة بوضوح
   ✅ الأزرار قابلة للضغط
   ✅ النصوص واضحة وقابلة للقراءة
   ✅ الحقول سهلة التعبئة
   ✅ لا يوجد scroll أفقي
   ✅ الصور تتناسب مع الشاشة
```

#### على الجهاز اللوحي 📱
```
1. اختر iPad Pro من DevTools
2. تحقق من نفس النقاط السابقة
3. تأكد من استغلال المساحة بشكل جيد
```

#### على الكمبيوتر 💻
```
1. اختر Responsive من DevTools
2. جرب عدة أحجام: 1024px, 1280px, 1920px
3. تحقق من التصميم الكامل
```

```

2. اذهب لتبويب Network

   ✅ وقت التحميل الكلي < 3 ثواني
   ✅ حجم الصفحة معقول
```
