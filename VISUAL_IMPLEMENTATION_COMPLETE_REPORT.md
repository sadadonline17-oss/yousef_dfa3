## 9. Visual Design System

### Color Palette (Example: SADAD)
```typescript
{
  primary: '#F58220',      // SADAD Orange
  secondary: '#E67317',    // Darker Orange
  accent: '#FFFFFF',       // White
  background: '#FFFFFF',   // Pure White
  surface: '#FFF8F2',      // Light Orange Tint
  text: '#1A1A1A',         // Near Black
  textLight: '#666666',    // Gray
  textOnPrimary: '#FFFFFF',// White on Orange
  border: '#FFE5D0'        // Light Border
}
```

### Typography
- **Arabic:** Cairo, Tajawal, sans-serif
- **English:** Inter, system-ui, sans-serif
- **Sizes:** text-xs (10px) to text-3xl (30px)
- **Weights:** font-normal, font-semibold, font-bold

### Shadows
```typescript
{
  sm: '0 1px 2px 0 rgba(..., 0.08)',
  md: '0 4px 6px -1px rgba(..., 0.15)',
  lg: '0 10px 15px -3px rgba(..., 0.20)'
}
```

### Border Radius
```typescript
{
  sm: '6px',   // Small inputs
  md: '10px',  // Cards
  lg: '14px'   // Large containers
}
```
